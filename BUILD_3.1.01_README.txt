Build 3.1.01 - First UI Development (UI scaffolding)

This drop includes:
- Top bar (hamburger + centered title)
- Bottom tabs: Home, Listings, Add, Compare, Calendar (no floating buttons)
- Left slide-out panels: Menu + Filters
- Right slide-out panel: Listing Details
- Listings cards with action buttons + preferred heart + status pill
- Manual sorting: long-press drag reorder WITHIN group (Preferred group and Other group are separate lists)

NOTE: Requires dependency:
- react-native-draggable-flatlist

If not installed, install from repo root:
npm i react-native-draggable-flatlist

Then restart Expo:
npx expo start --tunnel -c
